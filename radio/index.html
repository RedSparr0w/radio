steam folder/steamapps/common/Counter-Strike Global Offensive/csgo/resource/ui/

Create Custom Commands

The "label" text box is the label for the entry.
This may contain spaces.

The "Command" text box is the actual CS:GO Console command that will be used.

Example: 

voice_enable 0

Multi-line commands work!
In the command box, enter your commands in this format: 

sv_cheats 1;cl_drawhud 0;noclip
or click the format button and we will do it for you!

Do not enter alias/bind commands which activate multiple actions at once like so: 

bind "h" "+jump;-attack"
as the quote marks are removed when added, so your command will not execute as intended
