Options:
Command Creator:


  • The "Name" text box represents the name that
    will appear in the unused box up at the top.

  • The "label" text box is the label for the entry.
    This must NOT contain any spaces.

  • The "Command" text box is the actual
    CS:GO Console command that will be used.
    Example: "sv_cheats 1;cl_drawhud 0;noclip"
